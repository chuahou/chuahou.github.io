FGO AP Calculator

A lightweight AP calculator and reminder app for Fate/Grand Order

 GitHub |  Google Play

A simple app that calculates when AP will be refilled to:

  1. the desired level, and
  2. the maximum capacity.

The Google Play release is in English and contains no Fate graphics. My personal version in Japanese with Saber graphics can be downloaded from the GitHub page.

Mapo Tofu

A simple mapo tofu eating game made with Unity

 GitHub

This is a game about eating mapo tofu and managing spice that may or may not be loosely based on a minigame from a certain fighting game. The main purpose of creating this project is to familarise myself with mobile game development with Unity, so please do not expect a well-organised or well-written code base.

Brainfuck interpreters

Yet more brainfuck interpreters written in C++ and Rust.

 GitHub (C++ version) |  GitHub (Rust version)

Simple brainfuck interpreters written as an exercise in both C++ and Rust. Both repositories can be cloned and easily built using GNU make and Cargo respectively.

CHFace

Personalised custom face for Fitbit Versa.

 GitHub |  Fitbit Gallery

A custom watch face for my personal use that is unimaginatively named CHFace. As every watch face I could find did not match exactly what I wanted, I made my own.

As it is made for my personal use, it currently has very little in the way of customisation options and way too much in the way of hardcoded values.
